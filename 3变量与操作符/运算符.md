## 比较运算符 比较的是 值是否相等
## 身份运算符比较的是 内存地址是否相等
a = 1;
b = 2;
c = 1;
d = 1.0

a is b  ===>> false
a is c  ===>> true
a is d  ===>> false
比较的是内存地址

例题:
 a = {1,2,3}
 b = {2,1,3}
 a == b     true   比较的是值 集合是无序的 不管位置是否相等 只比较值
 a is b     false  比较的是内存地址 内存地址不同 肯定是false


 c = (1,2,3);
 d = (2,1,3);
 c == d     false  元组属于序列 跟位置是有关系的
 c is d     false
