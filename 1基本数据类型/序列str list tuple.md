# 在python中, str list tuple 统称为序列

共有的函数:

  len([1,2,3]) 函数 ====>>> 返回长度  3
  max([1,2,3]) 函数 ====>>> 返回最大值 3
  min([1,2,3]) 函数 ====>>>     最小值 1

相同点:
  在序列中的每个元素, 都有一个序号(index), 可以通过序号取值
  切片操作 [1,2,3][1:2] = [2,3]
  序列可以 + * 操作
  

判断一个序列中, 是否包含某个元素   in关键字 返回bool类型
  3 in [12,4,5,6];

判断一个序列中 是否 不包含 某个元素  not in 
  3 not in [12,4,5];  ==> true
